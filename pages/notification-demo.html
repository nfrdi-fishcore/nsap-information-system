Usage in Your Code

Include the notification files in your HTML:

<link rel="stylesheet" href="notifications.css">
<script src="notifications.js"></script>

Then use the global toast object:

// Simple usage
toast.success('Message here');
toast.error('Error message');
toast.warning('Warning message');
toast.info('Info message');

// With custom title
toast.success('Message', 'Custom Title');

// Advanced usage
toast.show({
    type: 'success',
    title: 'Custom Title',
    message: 'Your message',
    duration: 3000,
    closable: true
});
